l# Driver Drowsiness Detection

This project monitors driver drowsiness using **OpenCV** and **MediaPipe**, and alerts if eyes are closed or yawning is detected. It also sends snapshots to a Flask server.

## Requirements

- Python 3.8+
- OpenCV, MediaPipe, pygame, requests, Flask, numpy (see `requirements.txt`)
